## ðŸž Reporting a Vulnerability

**DO NOT** open a public GitHub issue for security vulnerabilities.

If you discover a security issue, please email our Security Team at:

**security@orpheon.protocol**

Please include:
*   Description of the vulnerability.
*   Steps to reproduce.
*   Potential impact (e.g., Remote Code Execution, Intent Spoofing, Ledger Corruption).

We will acknowledge your report within 24 hours and aim to provide a fix within 72 hours.
